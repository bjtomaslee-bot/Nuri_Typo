# Nuri Typography (누리 타이포그래피)

## 프로젝트 개요
Nuri Typography는 국제음성기호(IPA)의 개념을 한글 체계에 접목하여, 기존의 표준 한글 타이포그래피 엔진으로는 표기할 수 없는 새로운 형태의 글자와 음성학적 기호를 작성하기 위한 특수 목적 타이포그래피 프로그램입니다.

본 프로젝트는 한글의 제자 원리를 해체하고 재조합함으로써, 언어학적 연구나 특수 표기법이 필요한 환경에서 유연하게 대응할 수 있는 한글 조판 엔진을 제공하는 것을 목적으로 합니다.

---

## 주요 기능 및 특징

### 1. 비정형 한글 조합 엔진 (Phonetic Composition Engine)
- 표준 유니코드 한글 조합의 한계를 넘어, 기존 상용 폰트에서는 지원하지 않는 **'아래아(ㆍ)'** 같은 고어 요소나 **'ㄹㄹ', 'ㄴㄷ'** 과 같이 현대 국어의 표기 규칙을 벗어나는 비정형 겹자음/모음도 제약 없이 결합할 수 있습니다.
- 입력된 문자를 18가지 이상의 마스터 유형으로 자동 분류하는 알고리즘을 통해 체계적인 조판 환경을 제공합니다.

### 2. 정밀 수치 제어 (Precision Typography Control)
- 각 자소의 위치(X, Y), 장평(Width), 높이(Height), 굵기(Weight)를 개별적으로 제어하여 미세한 음성학적 차이를 시각적으로 표현합니다.
- 키보드 방향키를 이용한 픽셀 단위 넛지(Nudge) 기능을 통해 정밀한 보정 작업을 지원합니다.

### 3. 클라우드 기반 협업 환경
- Firebase Firestore를 활용하여 생성된 특수 글리프(Glyph) 데이터와 유형 설정값을 실시간으로 동기화하고 공유합니다.
- 익명 인증 기반의 협업 시스템을 통해 별도의 복잡한 절차 없이 프로젝트 데이터를 관리합니다.

### 4. 전문 벡터 데이터 추출
- 생성된 결과물을 투명 배경의 PNG 및 벡터(SVG) 형식으로 즉시 추출하여 외부 디자인 툴이나 학술 문서에서 활용할 수 있습니다.
- 설정된 배치 값을 JSON 코드 스니펫으로 추출하여 데이터의 재사용성을 확보합니다.

## 기술 스택

- **프레임워크**: React (v19), Vite
- **상태 관리**: useReducer 기반의 통합 상태 제어
- **백엔드**: Firebase Firestore, Auth
- **그래픽**: Canvas API 및 SVG 렌더링 엔진
- **폰트**: Noto Serif KR (Variable Font 지원 환경 최적화)
